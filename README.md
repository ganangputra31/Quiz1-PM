## Penjelasan Pengaruh NIM

Pada program ini, 3 digit terakhir dari NIM digunakan sebagai nilai unik yang dimasukkan ke dalam list harga barang.

Contoh:
NIM = 244107060008

3 digit terakhir = 008
Nilai tersebut dikonversi menjadi double yaitu 8.0.

Nilai ini ditambahkan ke dalam List<double> hargaBarang sehingga total belanja setiap mahasiswa akan berbeda sesuai dengan NIM masing-masing.
